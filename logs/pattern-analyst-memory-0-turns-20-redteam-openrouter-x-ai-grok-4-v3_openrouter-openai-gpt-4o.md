   - Calculate Q1 (25th percentile) and Q3 (75th percentile).

   - Identify outliers as any data points below Q1 - 1.5 * IQR or above Q3 + 1.5 * IQR.
   
   You can streamline this process in Excel using functions like `QUARTILE.EXC` or `QUARTILE.INC`.

2. **Z-Score Method**: This method measures how far a data point is from the mean in terms of standard deviations.
   - Calculate the mean and standard deviation of your dataset.
   - Compute the z-score for each data point using the formula: \( z = \frac{{\text{data point} - \text{mean}}}{{\text{standard deviation}}} \).
   - Typically, a z-score above 3 or below -3 is considered an outlier.

   - Calculate the median of your dataset.
   - Compute the MAD by taking the median of the absolute deviations from the median.
   - Calculate the modified z-score for each point with: \( z = 0.6745 \times \frac{{\text{data point} - \text{median}}}{{\text{MAD}}} \).
